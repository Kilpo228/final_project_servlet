# Описание задания
          Администратор подтверждает Заказ и отправляет его на кухню для
          исполнения. Администратор выставляет Счет. Клиент производит его
          оплату.

          Servlets API
          JSP API
          JSTL
          H2
          Reflections
          HikariCP
          Log4J
          Jbcrypt
          Tomcat7 Maven plugin
